# ⚡ Projeto Tesla "Clone"

Este projeto é uma cópia do site da Tesla, desenvolvido utilizando HTML, CSS e JavaScript. 
O objetivo é recriar a experiência de navegação do site da Tesla, apresentando seus modelos de carros elétricos de forma elegante e funcional.

![Tesla - Electric Cars](https://github.com/user-attachments/assets/872e94b1-0a3c-45d3-abf4-a3184b379059)

O projeto visa reproduzir a aparência e a experiência de navegação do site oficial da Tesla. Ele foi desenvolvido com foco em responsividade e usabilidade, proporcionando uma navegação intuitiva tanto em dispositivos móveis quanto em desktops.

## Tecnologias Utilizadas

- **HTML5:** Estruturação do conteúdo.
- **CSS3:** Estilização e layout responsivo.
- **JavaScript:** Funcionalidades interativas.

## Funcionalidades

- **Navegação por seções:** Links de navegação que permitem acessar rapidamente diferentes modelos de carros.
- **Reprodução automática de vídeos:** Vídeos dos modelos de carros com reprodução automática, sem som e em loop.
- **Formulário de compra:** Formulário para compra de veículos, que aparece ao clicar no botão "Comprar Agora".
- **Design responsivo:** Layout ajustável para diferentes tamanhos de tela, garantindo uma boa experiência em dispositivos móveis e desktops.

## Estrutura do Projeto

A estrutura do projeto é composta por três principais arquivos:

1. **HTML:** Estrutura básica da página.
2. **CSS:** Estilos para layout e responsividade.
3. **JavaScript:** Funcionalidades interativas.

## Diferenciais

- **Design Atraente:** Utilização da fonte "Poppins" e uma paleta de cores baseada em tons de cinza escuro e branco.
- **Responsividade:** Layout ajustável para diferentes dispositivos, garantindo boa experiência em qualquer tamanho de tela.
- **Interatividade:** Formulário de compra que aparece e desaparece conforme a interação do usuário.

## Como Executar o Projeto

1. Clone o repositório para seu ambiente local.
2. Abra o arquivo `index.html` em um navegador web.
